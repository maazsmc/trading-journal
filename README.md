# Trading Journal Desktop App

A modern, responsive, and standalone Windows application designed for traders to track their performance, analyze their strategies, and improve their trading discipline.

---

## Key Features

- **Standalone Windows App**: No browser or extra software needed. Just install and trade.
- **Comprehensive Logging**: Record every trade detail including entry/exit prices, position size, and strategy.
- **Visual Analytics**: Interactive charts showing your P&L, win rate, and performance by currency pair.
- **Local Data Persistence**: Your data is stored securely on your computer and persists across sessions.
- **Clean UI**: A focused, distraction-free interface with light and dark mode support.
- **Risk Management**: Track your risk/reward ratios and maintain disciplined position sizing.

---

## How to Install

1. Go to the [Releases](https://github.com/maazsmc/trading-journal/releases ) section of this repository.
2. Download the latest **`TradingJournalSetup.exe`** file.
3. Run the installer on your Windows system.
4. The app will automatically install, create a desktop shortcut, and launch.

---

## Built With

- **Electron**: For the desktop application wrapper.
- **Vanilla JavaScript**: For core logic and data management.
- **Bootstrap 5**: For a responsive and modern UI.
- **Chart.js**: For beautiful, interactive data visualizations.
